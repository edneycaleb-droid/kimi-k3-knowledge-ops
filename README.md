# Kimi K3 Knowledge Ops

A read-only research repository for tracking official Kimi K3 sources, availability, tools, API surfaces, release milestones, provenance, and security findings.

## Current verified baseline

- Launch announced: July 16, 2026
- Weights scheduled by Moonshot: July 27, 2026
- Current access surfaces: Kimi, Kimi Work, Kimi Code, and the Kimi API

Dates and availability are tracked as claims with first-party provenance and should be revalidated by scheduled research updates.

## Safety boundary

Discovered code is never automatically executed, installed, imported, or enabled.

## Layout

- `sources/official.yml` — first-party source inventory
- `catalog/` — tools, API, product surfaces, skills, and MCP-related records
- `releases/` — launch, weights, and subsequent release tracking
- `provenance/` — source receipts and integrity metadata
- `security/` — screening policy and findings
- `.github/workflows/research-update.yml` — scheduled read-only validation
